== Installation ==

1. Upload the plugin files to the /wp-content/plugins/ directory, or upload the .zip file with the plugin under **Plugins > Add New > Upload.**
2. Activate the plugin through the ‘Plugins’ screen in WordPress
3. Navigate to **Elementor -> AI Data Science Templates for Elementor** to import templates
